=============Getting matching JPA Persistence Libraries====================

Download Hibernate Core 3.2.0, Hibernate Annotation and Hibernate Entitity Manager from the Hibernate site at www.hibernate.org, and HSQLDB from www.hsqldb.org

From the "Hibernate Core" copy the following to the libs directory; 

hibernate3.jar
lib/antlr-2.7.6rc1.jar
lib/asm-attrs.jar
lib/asm.jar
lib/cglib-2.1.3.jar
lib/commons-collections-2.1.1.jar
lib/commons-logging-1.0.4.jar
lib/dom4j-1.6.1.jar
lib/ehcache-1.1.jar
lib/jta.jar                                             

From the "Hibernate Annotations", copy the following to the libs directory;
hibernate-annotations.jar

From the "Hibernate EntityManager", copy the following to the libs directory;

hibernate-entitymanager.jar
lib/javassist.jar
lib/jboss-archive-browsing.jar
lib/ejb3-persistence.jar

Finally from the HSQLDB archive, copy hsqldb.jar to the libs directory.
